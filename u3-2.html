บทที่ 3: ใช้อินเทอร์เน็ตในการค้นหาข้อมูลอย่างมีประสิทธิภาพ

โปรแกรมค้นหา (Search Engine)

ค้นหาโลกแห่งข้อมูลด้วยเครื่องมือมหัศจรรย์ของอินเทอร์เน็ต

Search Engine คือ โปรแกรมที่ออกแบบมาเพื่อสืบค้นข้อมูลบนโลกอินเทอร์เน็ต โดยเราต้องกรอกข้อมูลหรือคำที่ต้องการสืบค้นลงไป ซึ่งคำนี้เรียกว่า Keyword (คีย์เวิร์ด) จากนั้นระบบจะประมวลผลลัพธ์ที่เกี่ยวข้องออกมาให้เราเลือกใช้งาน

ตัวอย่าง Keyword:

"วิธีเลี้ยงปลาทอง", "สูตรขนมเค้กช็อกโกแลต", "สถานที่ท่องเที่ยวจังหวัดเชียงใหม่"

การค้นหาข้อมูลคืออะไร?

คือการสืบค้นสิ่งที่ต้องการทราบบนเครือข่ายอินเทอร์เน็ตผ่าน Search Engine Site ซึ่งทำหน้าที่รวบรวมรายชื่อเว็บไซต์ต่างๆ ไว้เป็นหมวดหมู่ ทำให้ประหยัดเวลาและได้รับผลลัพธ์ที่แม่นยำกว่าการหาด้วยตนเอง

Search Engine Site ที่นิยม

Google

อันดับหนึ่งของโลกและประเทศไทย มีความแม่นยำสูง

www.google.com
จุดเด่น: รวดเร็วและครอบคลุมที่สุด

Bing

พัฒนาโดย Microsoft เป็นคู่แข่งหลักของ Google

www.bing.com
จุดเด่น: รูปภาพพื้นหลังสวยงามทุกวัน

Yahoo

เน้นการนำเสนอข่าวสารและบริการอีเมลไปพร้อมการค้นหา

www.yahoo.com
จุดเด่น: ข่าวสารอัปเดตตลอดเวลา

DuckDuckGo

หน้าตาสะอาด ปลอดภัย และไม่ค่อยมีโฆษณา

duckduckgo.com
จุดเด่น: เน้นความเป็นส่วนตัวสูง

ประโยชน์ของ Search Engines

ช่วยให้สืบค้นข้อมูลได้อย่างสะดวก รวดเร็ว และมีประสิทธิภาพ โดยมีข้อดีหลักๆ ดังนี้:

ค้นหาเว็บไซต์ที่ต้องการได้รวดเร็ว

เพียงพิมพ์คำค้นหา ก็ได้ผลลัพธ์ภายในไม่กี่วินาที

ค้นหาข้อมูลได้หลากหลาย

ข่าวสาร, เพลง, รูปภาพ, วิดีโอ, แผนที่

ค้นหาข้อมูลจากเว็บไซต์เฉพาะทางได้

เช่น ข้อมูลการเรียน การแพทย์ กฎหมาย

รองรับการค้นหาได้หลายภาษาทั่วโลก

ค้นหาข้อมูลภาษาอังกฤษ ภาษาจีน ภาษาญี่ปุ่น ฯลฯ

เกร็ดความรู้:

ในปัจจุบันการสร้างเว็บไซต์ควรคำนึงถึงรูปแบบที่ Search Engine ค้นหาได้ง่าย ซึ่งเรียกว่าการทำ SEO (Search Engine Optimization)

SEO คืออะไร? เทคนิคทำให้เว็บไซต์ติดอันดับต้นๆ ของผลการค้นหา

ลองค้นหาดูสิ!

เปิดเบราว์เซอร์และลองค้นหาข้อมูลต่อไปนี้:

ค้นหารูปภาพ

"ลูกสุนัขพันธุ์ชิวาวา"

ค้นหาสูตรอาหาร

"วิธีทำไข่เจียวฟู"

ค้นหาข้อมูลเรียน

"ระบบสุริยะมีดาวเคราะห์กี่ดวง"

เคล็ดลับ: ใช้เครื่องหมายคำพูด (" ") เพื่อค้นหาคำที่ตรงเป๊ะ

ข้อมูลอ้างอิง

รอมฎอน สุทธิการ "เสิร์ชเอนจิน (Search Engines) คืออะไร หลักการทำงานของระบบค้นหาข้อมูล"

Search Engine ทำงานอย่างไร?

1

Crawling
รวบรวมข้อมูลจากเว็บไซต์

2

Indexing
จัดหมวดหมู่ข้อมูล

3

Ranking
จัดอันดับผลการค้นหา
